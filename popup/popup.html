About BrickTip!

Lego Sets Extension created by M.Nijdam. Displays a tooltip with set information on every number on a page. Found numbers are listed on the popup for easy reference.

For questions and/or suggestion concerning this extension please send an email to lego-extension@nijdam.nu
There will be absolutely no guarantee that your email will be answered, but i'll try to do so.
The creator of this extension can't be hold responsible for any damage or harm caused by the use of this software. Please uninstall this extension and create a moc.
